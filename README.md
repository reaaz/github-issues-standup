# Github Issues to Standup Update
Some quick JavaScript to run in your console on the Github Issues page to provide a rich text update of your stories.

## Usage
1. Open up the Github Issues for your repository.
2. Filter to issues assigned to you for the correct milestone.
3. Open up your JavaScript console and paste in the code to execute.
4. Copy the resulting HTML into your document of choice.
